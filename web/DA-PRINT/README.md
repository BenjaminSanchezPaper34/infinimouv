# 🎨 DA PRINT — Infini Mouv

Dossier dédié à la **direction artistique print** validée du client.
Dépose ici tout ce qui sert de référence visuelle pour la refonte web.

## Où déposer

➡️ Place tes fichiers dans **`A_DEPOSER/`** (exports PDF/PNG/JPG des affiches,
flyers, gourde, tapis, stories, posts, couverture, charte couleurs/typo…).

## Ce qui est utile

- **Exports aplatis** (PDF/PNG haute déf) des visuels InDesign — pour que je
  reproduise fidèlement la mise en page, les couleurs et la typo.
- **Charte** si elle existe (codes hex/CMJN, polices, règles d’usage du logo).
- **Logos** vectoriels (SVG/AI) versions couleur + monochrome blanc/noir.
- **Photos** sources haute résolution (salle, équipements, ambiance).

## Déjà intégré au site

- Vert signature : `#006935`
- Polices : Gotham (display, italique) + SF Pro Display (corps)
- Logos & photos webp → `public/images/`

> Une fois tes fichiers déposés, dis-le-moi : je les analyse et j’aligne le
> design system (`/design-system`) + les pages dessus.
